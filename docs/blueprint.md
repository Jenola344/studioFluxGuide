# **App Name**: FluxGuide

## Core Features:

- Simplified Trade: Intuitive trading interface with token selection and slippage control.
- Real-Time Charts: Clear display of market price charts powered by TradingView.
- Analytics Dashboard: Easy-to-use dashboard for viewing portfolio analytics, including P&L.
- Contextual DeFi Guide: AI-powered educational tool tool providing contextual help and explanations of DeFi concepts within the app.

## Style Guidelines:

- Primary color: Soft periwinkle blue (#9CB4CC) to evoke trust and sophistication in the DeFi space.
- Background color: Light, desaturated blue (#D3E4ED) to provide a clean, unobtrusive backdrop.
- Accent color: Warm apricot (#EEA47F) to draw attention to key interactive elements and CTAs.
- Font: 'Inter' (sans-serif) for a modern, clean, and highly readable user interface. Used for both headlines and body text.
